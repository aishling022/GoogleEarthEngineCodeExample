Annual and Monthly Composites Download from Landsat satalite images

Code included in files is used in code terminal: https://code.earthengine.google.com/

Project needs to be created before 

The code creates tasks to download composites in task manager in right window of screen

Composites are downloaded into folder in Google Drive

Code includes some preprocessing for use in python (changing the names of landsat bands, masks for clouds etc.)
